# 大型语言模型的上下文记忆受提示影响

[Machlab 和 Battle（2024）的新研究](https://arxiv.org/abs/2404.08865) 分析了不同大型语言模型在"大海捞针"测试中的上下文记忆性能。

## 关键发现

- 各类大语言模型在回忆事实时，受文本长度和信息位置的影响程度各有差异
- 模型的回忆性能对于提示中的细微调整表现出高度敏感性
- 提示内容与训练数据之间的交互作用可能导致回答质量下降

## 提升回忆能力的途径

可以通过扩大模型规模、改进注意力机制、探索不同的训练策略以及进行微调来提升模型回忆能力。

## 实践建议

通过持续评估，可以更深入了解各类 LLM 在具体应用场景中的表现，进而做出更优选择。审慎地设计提示、建立持续评估机制，并对不同的模型改进策略进行测试，对于提升记忆能力和实际效果至关重要。
