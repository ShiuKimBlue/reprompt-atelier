# Prompt Hub

Prompt Hub 是一个按任务类型分类的提示模板集合，帮助用户快速找到适合特定场景的提示示例。

本章涵盖 12 个分类的提示模板：

- **分类（Classification）**：情感分类、少样本情感分类
- **编程（Coding）**：代码片段、MySQL 查询、TiKZ 图表
- **创意（Creativity）**：押韵、跨学科创意、创造新词汇
- **评估（Evaluation）**：LLM-as-a-Judge 评估模式
- **信息提取（Information Extraction）**：从论文摘要中提取模型名称
- **图像生成（Image Generation）**：多轮迭代 TiKZ 绘图
- **数学（Mathematics）**：复合函数求值、奇数求和
- **问答（Question Answering）**：封闭域、开放域、科学问答
- **推理（Reasoning）**：间接推理、物理推理
- **文本摘要（Text Summarization）**：一句话概括
- **真实性（Truthfulness）**：幻觉识别
- **对抗性提示（Adversarial Prompting）**：提示词注入、泄露、越狱
