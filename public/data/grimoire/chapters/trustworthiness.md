# 大语言模型的可信度探索

在健康和金融等高风险领域开发应用时，建立可靠的 LLM 至关重要。虽然 ChatGPT 等模型能生成易读的回答，但在真实性、安全性、隐私保护等维度上无法保证可靠性。

[Sun 等人 2024 年的研究](https://arxiv.org/abs/2401.05561) 对 LLM 可信度问题进行了全面讨论，涵盖挑战、基准测试、评估方法、方案分析及发展方向。

## 可信度维度

作者定义了包含以下八个关键维度的评估框架：

| 维度 | 关键问题 |
|------|----------|
| 真实性 | 训练数据中的噪声、误导信息或信息过时导致挑战；接入外部知识来源的模型在真实性方面有所提升 |
| 安全性 | 开源模型在防止恶意利用、减少有害内容生成等方面通常不及专有模型 |
| 公平性 | 多数 LLM 在避免刻板印象方面不够理想，即使 GPT-4 准确率也仅约 65% |
| 鲁棒性 | 面对开放式问题或非典型数据分布时表现出较大不确定性 |
| 隐私 | 模型对个人敏感信息处理表现不一，部分模型在 Enron 邮件数据集测试中出现数据泄露 |
| 伦理观 | LLM 对道德原则有基本理解，但处理复杂伦理判断时往往力不从心 |

## 关键发现

- 专有 LLM 在可信度方面通常优于开源模型，但部分开源模型正逐渐缩小差距
- GPT-4 和 Llama 2 能有效拒绝刻板印象陈述，对抗性攻击展现出更强抵抗力
- Llama 2 等开源模型在未使用特殊内容审查工具时，可信度与专有模型相近
- 部分模型如 Llama 2 在追求高可信度时，可能牺牲特定任务性能，并将无害输入误判为有害内容
