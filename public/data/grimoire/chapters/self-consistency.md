# 自我一致性（Self-Consistency）

自我一致性是一种较为高级的提示工程技术，由 Wang 等人（2022）提出。该方法旨在"替换链式思维提示中使用的天真贪婪解码方法"。

核心思路是通过少样本 CoT 采样多条不同的推理路径，然后利用生成结果选出最一致的答案，从而提升 CoT 提示在算术和常识推理任务上的表现。

## 示例：算术推理

### 基础提示（无 CoT）

**提示：**

```
当我6岁时，我的妹妹是我的一半年龄。现在我70岁了，我的妹妹多大？
```

**输出：**

```
35
```

这个输出是错误的。

### 使用自我一致性改进

采用 Wang 等人 2022 年的少样本范例，提示包含多个示例：

```
Q：林中有15棵树。林业工人今天将在林中种树。完成后，将有21棵树。林业工人今天种了多少棵树？
A：从15棵开始，后来有21棵，差异即为种树数量：21-15=6棵。

Q：停车场有3辆汽车，又来了2辆，停车场有多少辆汽车？
A：3+2=5辆汽车。

Q：Leah有32块巧克力，姐姐有42块，吃了35块后还剩多少？
A：32+42=74块，吃了35块，剩余74-35=39块。

Q：Jason有20个棒棒糖，给Denny一些后剩12个，给了多少？
A：20-12=8个。

Q：Shawn有五个玩具，圣诞节从父母各得两个，现在有多少？
A：5+2=7，7+2=9个玩具。

Q：服务器房间有9台计算机，从周一到周四每天安装5台，现在有多少？
A：4天×5=20台，9+20=29台。

Q：Michael有58个高尔夫球，周二丢23个，周三丢2个，还剩多少？
A：58-23=35，35-2=33个球。

Q：Olivia有23美元，以每个3美元买了5个百吉饼，还剩多少？
A：花费3×5=15美元，剩余8美元。
```

### 采样多条推理路径

通过采样多条不同的推理路径，模型可能生成多个不同的答案：

**输出1：**

```
当我6岁时妹妹3岁（6/2）。现在70岁，妹妹70-3=67岁。
```

**输出2：**

```
叙述者6岁时妹妹3岁。现在70岁，妹妹70-3=67岁。
```

**输出3：**

```
当我6岁时妹妹3岁。现在70岁，妹妹70/2=35岁。
```

## 结论

计算最终答案涉及多个步骤，但从示例中可以看出，多数推理路径得出的答案是67，因此该答案将被选为最终答案。通过采样多条路径并取多数一致的结果，自我一致性有效纠正了朴素提示中常见的错误。
