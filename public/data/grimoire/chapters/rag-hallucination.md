# 利用 RAG 降低结构化输出中的虚假信息

ServiceNow 的研究人员发表了一篇[新论文](https://arxiv.org/abs/2404.08189)，探讨了如何为结构化输出任务高效部署 RAG 系统。

## 关键发现

- RAG 系统整合了小型语言模型和极小型检索器
- 该方法使得在资源受限环境下部署强大的 LLM 驱动系统成为可能
- 同时减轻了虚假信息问题并提升了输出的可靠性

## 应用场景

论文讨论了一种极具实用价值的企业应用场景，即把自然语言需求转换为 JSON 格式的工作流程。这一任务能够极大提升生产力，尽管仍有优化空间，例如"采用推测性解码或使用 YAML 代替 JSON"。

论文还提供了关于在现实世界中有效开发 RAG 系统的宝贵见解和实用建议。
