# 零样本提示

如今，经过大量数据训练并调整指令的LLM能够执行零样本任务。

## 示例

**提示：**

```
将文本分类为中性、负面或正面。

文本：我认为这次假期还可以。
情感：
```

**输出：**

```
中性
```

在上面的提示中，我们没有向模型提供任何示例——这就是零样本能力的作用。

## 指令调整与RLHF

指令调整已被证明可以改善零样本学习（Wei等人，2022）。其本质是在通过指令描述的数据集上对模型进行微调。此外，RLHF（来自人类反馈的强化学习）被采用来扩展指令调整，使模型更好地适应人类偏好，ChatGPT等模型正是这一发展的产物。

## 下一步

当零样本不起作用时，建议在提示中提供演示或示例，这就是少样本提示的由来。
